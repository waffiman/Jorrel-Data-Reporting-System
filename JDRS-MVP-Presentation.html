CALCULATOR PRINCIPLE

How the KPI is Calculated

Each HR manager works with simple "1" markers in Google Sheets, while JDRS handles all formulas and KPI calculations in the background.

  • Column G (New CV): every new candidate gets a "1". Sum(G) = total people in the system.
  • Column H (Screened): after 8-question screening, manager marks "1". Sum(H) = total screened prospects.
  • Main KPI: Recruitment Efficiency = (Sum(H) / Sum(G)) × 100.
  • Example: 790 screened / 1018 total = 77.6% efficiency.
  • Dashboard logic: for each system we compare Target vs Data, track day-to-day dynamic (+/-), and compute Target Accrued.
